SIM Price Scraper

Download and run the scraper on your computer to collect the latest SIM deal prices

Download Scraper

Works on Mac and Windows

How it works

1

Download & unzip

Click the button above. Your browser will download a zip file — unzip it to get the SIM Price Scraper folder.

2

Open the scraper

Mac: Double-click Run Scraper.command
Windows: Double-click Run Scraper.bat
A browser window will open with the scraper interface. First run will auto-install Python if needed.

3

Click "Run Scraper"

In the browser interface, click the Run Scraper button. You'll see live progress as it scrapes all UK mobile providers. This usually takes 2–5 minutes.

4

Upload the results

When it finishes, a data file is saved to your Desktop. Come back here, click "Upload Data" in the dashboard, and select the file.

Note: The scraper requires Python 3 to run. If you don't have it, the launcher will offer to install it automatically — just follow the prompts.
